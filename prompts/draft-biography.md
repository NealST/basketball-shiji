# 提示词：起草传记正文 · Prompt: Draft Full Biography

> 使用方法：逐节向AI发送提示词，将对应章节的大纲要点和研究素材填入。

---

## 总起草提示词（全文一次生成）

适合信息充足、希望快速出初稿的情况。

```
你是一位精通《史记》纪传体风格的中文作家，同时深谙NBA历史。

请根据以下大纲和研究素材，为【[球员姓名]】撰写一篇完整的篮球传记。

写作要求：
1. 文风定位：70%现代汉语可读性 + 30%《史记》古典文学风味
2. 避免口水话，追求简洁有力的叙事语言
3. 关键时刻使用场景化叙事，有画面感
4. 以"太史公曰"格式撰写结语评论
5. 全文约3500–5000字

章节结构：
一、人物概览（150–250字）
二、年表（表格形式，8–15条）
三、早年岁月（500–800字）
四、崛起之路（600–900字）
五、巅峰时代（1000–1500字）
六、低谷与转折（400–600字）
七、历史地位与遗产（600–800字）
八、太史公曰（150–250字）

以下是大纲和研究素材：

【大纲】
[粘贴 outlines/<player-slug>-outline.md 的内容]

【研究素材】
[粘贴 research/<player-slug>.md 的关键信息]
```

---

## 分节起草提示词

适合希望精细控制每一节内容质量的情况。

### 第三节：早年岁月

```
请为【[球员姓名]】的传记撰写"早年岁月"章节，字数约500–800字。

要求：
- 从家庭背景和成长环境切入
- 描述其与篮球结缘的经历
- 如有重要的早年转折点（如某次比赛、某位导师）重点叙述
- 文风偏叙事，可有细节感，兼顾现代可读性与古典气质

相关素材：
[粘贴该球员早年经历的研究笔记]
```

### 第五节：巅峰时代（最核心章节）

```
请为【[球员姓名]】的传记撰写"巅峰时代"章节，字数约1000–1500字。

要求：
- 以生涯最辉煌的时期为核心，叙述其统治联盟的故事
- 至少详细描写2–3场标志性比赛或关键时刻，采用场景化叙事（有现场感、有细节）
- 体现时代背景：当时的联盟格局、主要对手、队友关系
- 文风应最具张力，节奏紧凑，适当使用短句制造戏剧感
- 这是全文最重要的部分，请展现最好的叙事水准

相关素材：
[粘贴巅峰时期的研究笔记，包括关键比赛数据和事件]
```

### 第八节：太史公曰

```
请为【[球员姓名]】的传记撰写"太史公曰"结语，字数约150–250字。

要求：
- 以"太史公曰："开头
- 模仿《史记》中司马迁直接评论历史人物的语气
- 对人物的精神特质、历史意义作总结性评价
- 可以有哲思性、有史论性，避免简单重复正文内容
- 这是全文古典风味最浓的部分，但仍须清晰可读
- 结尾应有余韵，令人回味

该球员的核心特质与历史地位：
[粘贴关键评价要点]

整篇文章的主要观点：
[简述已写完的正文主要论点]
```

## 使用提示

- 初稿完成后，将其保存至 `articles/drafts/<player-slug>.md`
- 对不满意的章节，可以重新发送对应的分节提示词，要求重写
- 确保YAML frontmatter在文件开头已正确填写
- 数据性内容（冠军次数、MVP次数等）须与研究笔记核对
