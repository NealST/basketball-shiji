# 提示词：古典风格润色 · Prompt: Classical Style Rewrite

> 使用方法：将需要润色的段落或章节粘贴进提示词，要求AI进行古典风味改写。

---

## 整节润色提示词

```
你是一位精通《史记》文风的中文写作编辑。

请对以下文字进行润色改写，使其在保持现代可读性的前提下，增加约30%的古典汉语文学风味。

具体要求：
1. 保留原文的核心叙事内容和事实，不可增删关键信息
2. 适当将长句拆为短句，制造节奏感
3. 在适合的位置使用四字成语或成语典故
4. 关键时刻（如夺冠、绝杀）用更凝练、更有力的表达
5. 过于口语化的表达替换为更有文学感的措辞
6. 避免过度古文化——读者不需要查字典才能读懂

原文：
[粘贴需要润色的段落]
```

---

## "太史公曰"专项润色

```
请对以下"太史公曰"结语进行润色，使其充分体现《史记》评论风格。

要求：
1. 以"太史公曰："开头
2. 语气庄重、凝练，有历史评论的厚重感
3. 可适当使用古文句式（如"……者，……也"、"呜呼"等），但整体仍可读
4. 避免堆砌辞藻——每一句都应有实质内容
5. 结尾应有余韵，可用反问或感叹收束

当前版本：
[粘贴当前太史公曰内容]

该球员的核心特质（供参考）：
[简述2–3个核心特质]
```

## 对比参考：改写示例

### 改写前（过于口语）
> 乔丹在那场比赛里表现得特别好，他得了45分，带着球队赢得了比赛，全场球迷都沸腾了。

### 改写后（加入古典风味）
> 是夜，乔丹独得四十五分，运斤成风，无可匹敌。全场屏息，而后沸腾——此役之后，联盟知其不可撼也。

## 使用提示

- 不要一次性对整篇文章进行润色——分节进行，效果更好
- 润色后与原文对比，确认关键事实未被改变
- "太史公曰"节应是全文古典风味最浓的部分，可以进行2–3轮迭代
- 如果AI的改写过于文言、难以阅读，要求其"降低文言度，保持现代可读性"
