# 提示词：事实核查与一致性审查 · Prompt: Fact-Check & Consistency Review

> 使用方法：将完整文章发送给AI，要求进行事实与一致性核查。建议在发布前必做此步骤。

---

## 事实核查提示词

```
你是一位NBA历史专家和细致的文字编辑。

请对以下篮球传记文章进行全面的事实核查和一致性审查。

请检查以下内容，并以列表形式输出所有发现的问题或疑点：

【数据核查】
1. 冠军次数是否准确？
2. MVP次数（常规赛MVP、总决赛MVP）是否准确？
3. 全明星入选次数是否准确？
4. 关键赛季的场均数据（得分、篮板、助攻）是否大致准确？
5. 选秀顺位和选秀年份是否正确？
6. 重要比赛（如总决赛）的年份是否正确？

【事件核查】
7. 关键事件的年份是否准确？
8. 球队名称和转会信息是否正确？
9. 教练姓名和队友名字是否正确？
10. 重要比赛的对阵双方是否正确？

【一致性检查】
11. 文章中同一数据是否前后一致（如某处说"五次MVP"，别处是否也如此）？
12. 球员姓名的中英文译写是否统一？
13. 时间线是否存在矛盾？

请对每个问题按以下格式输出：
- 位置：[哪个章节/段落]
- 问题：[具体描述疑点]
- 建议：[如何修正或核实]

文章内容：
[粘贴完整文章]
```

## 快速一致性检查提示词（轻量版）

适合快速扫描，不需要深度事实核查时使用。

```
请快速审查以下文章的内部一致性，重点检查：
1. 同一数字或事实是否在文章中多次出现但说法不一
2. 时间线是否有明显矛盾
3. 球员姓名、球队名称、人名的拼写是否统一
4. 是否有明显的常识性错误

请以要点列表形式输出，只列出有问题的地方。

文章：
[粘贴文章]
```

## 关键数据参考来源

进行事实核查时，建议参考以下来源（需自行查阅）：

- **Basketball-Reference.com** — 权威数据来源，统计最全
- **NBA.com/history** — NBA官方历史记录
- **Naismith Memorial Basketball Hall of Fame** — 名人堂官网入选信息
- **维基百科（英文）** — 综合信息参考
- **各大赛季的官方记录** — 总决赛结果、MVP获奖等

## 常见易错数据备忘

| 类别 | 注意点 |
|------|--------|
| 冠军次数 | 区分"以球员身份夺冠"和"总计夺冠次数"（含教练/管理层） |
| MVP次数 | 常规赛MVP与总决赛MVP须分开列举 |
| 选秀顺位 | 历史上有些球员的选秀顺位常被误记 |
| 年份 | 注意NBA赛季跨年（如1990–91赛季，夺冠是1991年） |
| 球队变迁 | 历史上有球队迁城改名，需使用当时的正确名称 |
| 数据四舍五入 | 场均数据保留一位小数 |

---

## 使用提示

- 此步骤**必须在发布前执行**
- AI可能无法核查2023年后的数据，请自行核对
- 核查完成后，在文章frontmatter中将 `status` 从 `draft` 更新为 `revision`
- 所有修正应记录在研究笔记中，以便后续参考
