# 提示词：标题与社交媒体钩子 · Prompt: Title Options & Social Media Hooks

> 使用方法：在文章完成修订后，使用本提示词生成标题选项和各平台推广文案。

---

## 标题生成提示词

```
你是一位擅长中文内容创作的编辑，熟悉《史记》文风以及现代社交媒体传播逻辑。

请为以下篮球传记文章生成标题选项。

要求：
1. 生成至少6个标题选项，涵盖以下不同风格：
   - 古典史书风格（如《[人名]列传》《[朝代/时代]第一[位置]》）
   - 现代吸引力风格（有张力、有悬念、或有情感共鸣）
   - 混合风格（古典词汇+现代表达）
2. 每个标题附上一句适合作为副标题的短句（20字以内）
3. 标注每个标题适合的发布平台（微信公众号/Substack/微博/小红书）

球员姓名：[球员姓名]
文章核心主题：[一句话描述文章最想传达的主旨]
目标读者：[篮球迷 / 文化读者 / 通识读者]

文章摘要（可选，粘贴"太史公曰"或概览节）：
[粘贴内容]
```

## 社交媒体推广文案提示词

### 微信公众号 推文引言

```
请为以下传记文章撰写一段微信公众号推文引言，约80–120字。

要求：
- 开头吸引眼球，可以用问句、反转或强烈对比
- 中间简述文章的核心看点
- 结尾引导读者点击阅读全文
- 语气：有文化感，但不沉闷；适合在手机端阅读

文章标题：[标题]
核心内容：[一句话描述]
```

### 微博 / 短视频平台 摘要

```
请为以下传记文章生成一条适合微博或短视频平台配文的推广文案，约50–80字。

要求：
- 简洁有力，第一句必须抓人
- 可以包含1–2个话题标签（如 #NBA历史# #篮球传记#）
- 语气活泼，但不失文化品位

文章标题：[标题]
文章亮点：[2–3个具体亮点，如"首次用史记体写乔丹"、"还原1998年总决赛最后一投"]
```

### Substack / 邮件订阅 主题行

```
请为以下传记文章生成3个英文邮件主题行（Subject Line），适合Substack或英文邮件订阅列表。

要求：
- 每个主题行约40–60字符
- 风格1：引发好奇心
- 风格2：强调文化独特性（"Shiji-style biography"）
- 风格3：直接说明内容价值

球员：[球员英文姓名]
文章亮点：[用英文简述1–2个亮点]
```

## 使用提示

- 标题确认后填入文章frontmatter的 `title` 字段
- 推广文案保存在 `assets/` 目录下对应球员的文件夹（如有）
- 不同平台使用不同风格的文案，避免千篇一律
- 在微信公众号和小红书，标题可以适度加入emoji，增加视觉吸引力
